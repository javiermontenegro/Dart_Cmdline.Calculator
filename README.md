### Prerequisites

In order to be able to install cmdline calculator you will need following software:
Dart SDK >=2.10.0-2.0.dev <2.11.0

### Dart tools requirements
```
$ pub global activate webdev
$ pub global activate stagehand
```

### Get dependencies
```
$ pub get
```
## Running the program 
```
$ dart main.dart
```
